### Social Media Front-End 

* React.js
* Material-UI
* Context-API

### About:-

* You can Login/Register.
* You can share post.
* You can Like / Dislike post.
* You can chat with your friends.

####  To run this app in local server - ( Client ) 

```
git clone 
cd filename
npm install
npm start 
```
